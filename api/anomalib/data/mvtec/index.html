anomalib.data.mvtec¶

MVTec Dataset.

MVTec This script contains PyTorch Dataset, Dataloader and PyTorch Lightning DataModule for the MVTec dataset.

If the dataset is not on the file system, the script downloads and extracts the dataset and create PyTorch data objects.
